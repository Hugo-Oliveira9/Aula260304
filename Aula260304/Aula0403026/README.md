# Servidor e Rotas com Java, JSON e Python

## 📌 Sobre o Projeto
Este projeto demonstra a implementação básica de um **servidor com sistema de rotas**, utilizando **Java**, **JSON** e **Python** para manipulação de dados e comunicação entre aplicações.

O objetivo é apresentar conceitos fundamentais de:
- Criação de servidor
- Definição de rotas/endpoints
- Troca de dados utilizando JSON
- Integração entre diferentes linguagens

## 🚀 Tecnologias Utilizadas
- Java
- Python
- JSON

## ⚙️ Funcionalidades
- Inicialização de um servidor local
- Definição de rotas para requisições
- Retorno de respostas estruturadas em JSON
- Processamento de dados entre Java e Python
